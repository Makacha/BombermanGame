# Bomberman Game

Kill all enemies and find the portal to win!

## Game control

### Move:

- ***Up***: KEY_UP
- ***Down***: KEY_DOWN
- ***Left***: KEY_LEFT
- ***Right***: KEY_RIGHT
- ***Up & Left***: KEY_UP + KEY_LEFT
- ***Down & Left***: KEY_DOWN + KEY_LEFT
- ***Up & Right***: KEY_UP + KEY_RIGHT
- ***Down & Right***: KEY_DOWN + KEY_RIGHT

### Place Bomb:

- ***Place Bomb***: KEY_SPACE

### Other:
- ***Pause***: KEY_P
- ***Restart***: KEY_F5
